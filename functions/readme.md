# Funktionen

**Was ist eine Funktion?**

- Sammlung von Anweisungen
- werden erst beim Aufruf der Funktion ausgeführt
- Funktionen schaffen Struktur und Übersichtlichkeit
- Funktionen steigern die Performance
- Funktionen verringern Fehler
- Wiederverwendbarkeit -... bei guter Kapselung über Projekte hinweg wiederverwendbar
- Funktionen ermöglichen eine schnellere und bessere Anpassbarkeit
- Funktionen lohnen sich besonders, wenn bestimmter Code in einen Script mehrfach ausgeführt werden sollen
- Funktionen sind frei definierbar mit frei vergebenen Namen

Funktionen sind wie blackboxes
- oben rein
- unten raus
